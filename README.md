This project is part of the Recipe Finder Web App, built using Django.
It includes user registration using Django Models, Forms, and Views.

# Features
- User Sign-Up System using Django Forms
- Form validation (required fields, email format, password confirmation)
- Database integration using Django ORM
- CSRF-protected form submission
- Clean, well-structured URLs, views, and templates
- Static files for styling (CSS, JS)
